## Description
Garden of Wishes is a web-based platform that allows users to book a virtual or real visit to a flower garden.
The website provides a calm and interactive experience where users can choose flower types, select visit dates
and times, and submit their bookings online.

The system validates user input on both the front-end and back-end, stores data securely, and allows users
to view their bookings using their name and phone number.

## Features
- Responsive multi-page website
- Booking form with validation
- User input validation (HTML & JavaScript)
- Booking lookup page
- Clean and aesthetic UI design
